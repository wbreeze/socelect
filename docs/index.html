socelect

Find alternatives that satisfy most.

Socelect is a place where you can find out what people prefer. Any choice you can think of. You can find people's preferences here.

Your invited express their preference by ranked choice. The result is the ranking with maximum agreement.

This site is semi-permanantly down for maintenance.

If you would like to submit a proposal for hosting this site, please visit the socelect GitHub repository, review the code base, and use the contact information found there. Briefly, it is Ruby on Rails with React.js, requires a small database, uses the daemons gem to run background tasks, and requires compilation and installation of the Davenport library (written in the C language).
